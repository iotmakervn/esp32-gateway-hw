# esp32-gateway-hw

Gồm các module :

- 1 module ESP32-WROVER 
- 1 module GSM-SIM800C
- 1 module GPS-NEO6M
- Header nạp chương trình cho esp32
- Led báo nguồn, GPS, GSM
- Nguồn vào 12V :
	+ IC LM2596S
	+ ICMCP1727
- Có dùng pin 3v7 và sạc khi cắm nguồn
	+ IC MCP73832T-2ACI/OT
- Kích thước: để vừa hộp nhôm 100x75x38
